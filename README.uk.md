# Адресна книга - дні народження

CLI-адресна книга з відстеженням днів народження та запитом "у кого день народження протягом наступних 7 днів", з перенесенням вихідних (день народження у суботу/неділю показується на наступний понеділок).

## Стек

- Python 3.12 (тільки стандартна бібліотека)

## Команди

| Команда | Дія |
|---------|-----|
| `add <ім'я> <телефон>` | Створити контакт (валідація 10 цифр) |
| `change <ім'я> <старий_тел> <новий_тел>` | Оновити телефон |
| `phone <ім'я>` | Показати всі телефони контакту |
| `add-birthday <ім'я> <DD.MM.YYYY>` | Встановити день народження |
| `show-birthday <ім'я>` | Показати день народження контакту |
| `birthdays` | Показати дні народження на найближчі 7 днів |
| `all` | Показати всі контакти |
| `close` / `exit` | Вихід |

## Запуск

```bash
python hw7.py
```
